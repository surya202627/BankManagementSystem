# 🏦 Console-Based Bank Management System

A simple **Console-Based Bank Management System** developed using Java. This project demonstrates the use of Object-Oriented Programming (OOP) concepts to perform basic banking operations through a menu-driven interface.

---

## 📌 Objective

Develop a banking application that allows users to:

- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- View account details

---

## 🛠️ Technologies Used

- Java
- VS Code
- Terminal

---

## 📂 Project Structure

```
BankManagementSystem/
│── BankAccount.java
│── BankManagementSystem.java
│── README.md
```

---

## ✨ Features

- Create a new bank account
- Deposit money into an account
- Withdraw money with balance validation
- Check current account balance
- View account information
- Menu-driven console interface
- Input validation
- Exception handling for invalid operations

---

## 📚 Concepts Used

- Classes and Objects
- Encapsulation
- Constructors
- Methods
- Scanner Class
- Conditional Statements
- Switch Statement
- Loops
- Object-Oriented Programming (OOP)

---

## ▶️ How to Run

### 1. Compile the Java files

```bash
javac BankAccount.java BankManagementSystem.java
```

### 2. Run the application

```bash
java BankManagementSystem
```
